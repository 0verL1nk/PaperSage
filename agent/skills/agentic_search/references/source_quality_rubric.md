# Source Quality Rubric

Score each source on a 0-1 scale:

- credibility: authority and publication trust
- relevance: alignment with current sub-query
- recency: freshness for time-sensitive claims
- consistency: agreement with independent sources

Suggested weighted score:

`score = 0.35*credibility + 0.35*relevance + 0.15*recency + 0.15*consistency`

## Conflict Handling

- If high-score sources conflict, keep both claims and mark "conflict".
- If only low-score source supports a claim, mark "weak evidence".
- Never convert weak/conflicting evidence into definitive conclusions.
